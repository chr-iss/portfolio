# portfolio  
# 🌐 Cresjan Sithole – Developer Portfolio

This repository contains the source code for my personal portfolio website, designed to showcase my skills, projects, education, and experience as an aspiring Software Developer.

🔗 **Live Website:** https://cres-portfolio.netlify.app/

---

## 📖 About

My portfolio serves as a central place where recruiters, collaborators, and fellow developers can learn more about me and explore the projects I've built. It highlights my technical skills, development journey, and passion for creating software that solves real-world problems.

---

## ✨ Features

- Modern and responsive design
- About Me section
- Technical skills showcase
- Featured projects
- Education and experience
- Contact section
- Resume download
- Mobile-friendly interface
- Smooth animations and navigation

---

## 🛠️ Built With

- Git & GitHub

## 🚀 Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/chr-iss/portfolio.git

# Open the project folder
cd portfolio

# Launch index.html in your browser
```

Or simply visit the live site:

**https://cres-portfolio.netlify.app/**
